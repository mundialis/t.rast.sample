# t.rast.sample

Sample a space time raster dataset at specific vector point coordinates
and write the output into a file or stdout. This is a PyGRASS implementation.

## Installation

The installation requires [GRASS 8](https://grass.osgeo.org/).

```bash
grass --tmp-location XY --exec g.extension extension=t.rast.sample url=https://github.com/mundialis/t.rast.sample
```
